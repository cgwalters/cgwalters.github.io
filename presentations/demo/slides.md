# Demo Presentation

Marp-backed slides live in `presentations/` and are published as `/presentations/<deck>/`.

---

## Workflow

1. Edit `slides.md`
2. Run `just build demo`
3. Preview with `just preview demo` or let CI publish the built deck

---

## Local Preview

Use `just preview demo` for live reload while editing.
